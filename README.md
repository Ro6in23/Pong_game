# Pong_game
A classic two-player Pong game built with Python Turtle Graphics featuring ball physics, paddle controls, and score tracking.

# Features
- Two player paddle control
- Ball bouncing physics
- Score tracking
- Boundary detection

# Technologies Used
- Python 3
- Turtle Graphics

# Controls
| Right Paddle : |↑ Arrow | ↓ Arrow |
| Left Paddle  : | W | S |


# Project Structure
- `main.py` - Main game loop
- `ball.py` - Ball movement & bouncing
- `paddle.py` - Paddle movement & boundary
- `scoreboard.py` - Score tracking & display
